# 🎵 LyricLexicon

LyricLexicon is a full-stack web application that analyzes the vocabulary of songs. Users can search for a song and artist, automatically retrieve the lyrics, and explore detailed word statistics.

## Features

- 🔍 Search for songs by title and artist
- 🎶 Automatically fetch lyrics using the Lyrics.ovh API
- 📊 Count the frequency of every word used in a song
- 🔤 Display all words in alphabetical order
- 🏆 Highlight the Top 10 most-used words
- 📈 Show total word count and unique word count
- ⚡ Built with a React frontend and Node.js/Express backend
